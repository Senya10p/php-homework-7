ВНИМАНИЕ! 

Работа содержит:
главная страница index.php (содержит ссылки для перехода в галерею и гостевую книгу),
папка с файлами для галерееи gallery и папка с файлами для гостевой книги guestbook.
файлы для новостей article.php, news.php и текстовый файл с новостями news.txt.

в classes находятся файлы с классами: Article.php, News.php, View.php

В папке gallery находятся папка с изображениями images, файл с функциями functions.php, файл с классом Uploader.php,
файл для авторизации login.php, текстовый файл log.txt, файл с формой для загрузки изображений index.php, 
файл с галереей gallery.php, файл для загрузки изображений upload.php, файл для выхода logout.php

В папке guestbook находятся папка с классами classes (содержит два файла с класами: GuestBook.php и TextFile.php), 
текстовый документ с записями гостевой книги guestbook1.txt, главную страницу гостевой книги index.php, 
в которой есть форма для добавления записей в гостевую книгу.

в папке templates находятся шаблоны
